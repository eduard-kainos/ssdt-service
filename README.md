# Same spec, different team - express-server

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

```
Node, npm, git
```

### Installing

```
- Clone the project
- npm install in the project root
- node index.js
```
